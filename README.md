# PewPew (and probably PewPew 2) Docs!

Running on port 80

# Some structures:
 - UploadScore:
   - nickName - player nick
   - score - yes
   - time - replay time
   - hwID - device ID?
   - hwClass - device OS
   - hwModel - device model
   - mapName - path to `.lua` map
   - version - game version
   - hash - uknown
 - GetScores:
   - Game host: `pewpew-2.appspot.com`
